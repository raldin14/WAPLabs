Identify block vs. inline HTML elements

  • Block elements:
    • <div>
    • <p>
    • <h1> to <h6>
    • <hr>
    • <section>
    • <header>
    • <footer>
    • <nav>
    • <aside>
    • <article>
    • <ol> / <li>
    • <ul> / <li>
    • <form>
  • Inline elements:
    • <span>
    • <br>
    • <img>
    • <a>
    • <strong>
    • <i>
    • <input>
    • <textarea>
    • <button>
    • <select> / <option>
    • <label>
